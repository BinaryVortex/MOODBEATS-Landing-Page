## Getting Started

To run this project locally:

1. Clone the repo

   ```bash
   git clone https://github.com/BinaryVortex/MOODBEATS-Landing-Page.git
   ```

2. Open the project folder and open `index.html` in your browser:

   - Double-click `index.html`, or
   - Serve it with a local web server (recommended for some browsers):

     ```bash
     # using Python 3
     python -m http.server 8000
     # then visit http://localhost:8000
     ```
